# Phase N1: Native基盤構築（2-3週間）

## 概要
React Nativeプロジェクトの基盤構築、開発環境設定、基本認証機能の実装を行います。

## チケット一覧

### N1-01: React Nativeプロジェクト初期設定
**優先度**: 高  
**工数**: 1日  
**担当**: Native開発者

#### 作業内容
- React Native CLI環境構築
- Expo設定（必要に応じて）
- TypeScript設定
- ESLint/Prettier設定
- フォルダ構造設計

#### 完了条件
- [ ] React Nativeプロジェクトが作成されている
- [ ] iOS/Androidで正常にビルドできる
- [ ] TypeScript設定が完了している
- [ ] 開発環境が整っている
